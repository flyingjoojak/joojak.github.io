# 📃 1차 프로로젝트
---
## 📚 프로젝트 주제
이번 프로젝트는 경구약제의 이미지들을 전처리하고 딥러닝하여 경구약제들을 분류하는 모델을 만드는 프로젝트이다

### 📚 담당한 역할
모델의 학습 및 하이퍼 파라미터 수정을 담당하였다

## 📚 기간별 일지
### 📚 개인 작업
처음 일주일은 각각 개인이 모델을 만들어보기로 하여 개인 작업만 하였다.
10/21 : SSD 모델 구성 및 기본 베이스라인 작성
10/22 : YOLO8모델로 베이스라인 수정
10/23 ~ 10/25 : 기본 데이터를 활용하여 에폭 및 옵튜나를 활용하여 최적의 하이퍼파라미터를 찾아 각 리더보드에 끼치는 점수 확인
### 📚 팀플 작업
10/24 ~ 10/29 : 전처리가 된 데이터를 받아 YOLO8, YOLO11을 모델별로 사용해보고 앙상블 및 증강들을 사용하여 기존 점수와 비교
10/30 ~ 11/02 : 기존의 데이터도 같이 사용해보고 점수가 일반화와 과적합을 각각 따로 적용해서 점수를 비교
11/03 ~ 11/04 : 각각 다른 데이터들을 사용해서 2차학습을 하고 점수를 비교하고 최적의 파라미터 확인
